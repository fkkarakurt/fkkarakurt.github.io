Reconic
  • All-in-one reconnaissance tool for cybersecurity professionals and bug hunters.
  • Helps map, analyze, and secure digital infrastructures with multiple integrated scanning features.
  • Features include:
    • WHOIS Lookup: Retrieves domain registration details, including ownership and administrative contacts.
    • DNS Resolution: Resolves DNS records to uncover domains, subdomains, and configurations.
    • SSL/TLS Certificate Inspection: Inspects certificates for validity, expiration, issuer, subject, and configuration details.
    • HTTP Header Analysis: Captures and analyzes HTTP headers to identify security issues and misconfigurations.
    • Port Scanning: Detects open ports and exposed services to understand the target’s attack surface.
    • Subdomain Discovery: Identifies active subdomains using DNS zone methods, expanding visibility of a domain.
    • Directory Traversal: Searches for accessible directories, revealing sensitive information or hidden admin panels.
    • JavaScript File Enumeration: Lists JavaScript files for deeper analysis of potential vulnerabilities or data leaks.
    • Technology Scanner: Detects frameworks, libraries, and platforms used by the target system.
    • CVE Integration: Provides direct links to known vulnerabilities for identified technologies.

GenCrack
  • Demonstrates the use of a genetic algorithm to evolve a population of random strings towards a predefined target string.
  • Applies the principles of natural selection, where the fittest individuals are chosen to reproduce and form the next generation.
  • Population initialization starts with randomly generated strings.
  • Selection uses a simple tournament approach, choosing two random individuals and advancing the fitter one.
  • Crossover combines two parents to produce a child string.
  • Mutation introduces randomness, with each character having a chance to change based on a predefined mutation rate.
  • Elitism ensures that a percentage of the fittest individuals move directly to the next generation without alteration.
